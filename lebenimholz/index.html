Leben im Holz

Allgemein

Server

Minecraft Edition: Java

Minecraft Version: 1.12.2 Forge

Verantwortlich: LonelyChimo, DieSchwitzritze, LegionReddog, 5kWBassMachine


Das Modpack ist im TechnicLauncher und auf GitHub verfügbar.

Es ist durch Life in the Woods inspiriert, auf dem unser altes Modpack basierte. Das heißt es ist aus ziemlich allen Bereichen etwas dabei, unter anderem Mods wie
  • Technik:
    • Applied Energistics 2
    • Immersive Engineering
    • Simple Quarry
    • Sticky Redstone
    • Einige Tinkers Mods
  • Magie:
    • Botania
    • Thaumcraft
  • Bauen:
    • Bibliocraft
    • Builderswand
    • Carpenter
    • Chisel
    • Decocraft
  • Verschiedenes:
    • Biomes O Plenty
    • Pam's Harvestcraft
    • Rucksäcke
    • Tinkers Construct
    • Twilight Forest
Eine komplette Liste gibt es auf GitHub.
